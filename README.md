# Speech-Emotion-Recognition

About RAVDESS dataset
This is the Ryerson Audio-Visual Database of Emotional Speech and Song dataset, and is free to download. This dataset has 7356 files rated by 247 individuals 10 times on emotional validity, intensity, and genuineness. The entire dataset is 24.8GB from 24 actors, but weâ€™ve lowered the sample rate on all the files. In our model, we are taking the emotion (third identifier) in consideration only.
